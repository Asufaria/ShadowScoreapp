🎯 ShadowScore クイックスタート

📋 必要な環境

  • Python 3.8以上がインストールされていること
  • Windowsの場合: PowerShellまたはコマンドプロンプト

🚀 簡単起動（推奨）

start.bat をダブルクリックするだけ！

自動で環境構築からアプリ起動まで行います。

⚡ 手動起動

PowerShellまたはコマンドプロンプトで以下を実行：

pip install -r requirements.txt
python -m uvicorn main:app --reload

🌐 アクセス

起動後、ブラウザで以下にアクセス：

http://localhost:8000

📖 使い方

  1. プレイヤーA・Bの名前を入力
  2. 先行プレイヤーを選択
  3. 対戦開始ボタンをクリック
  4. レアリティボタンでポイントを追加
  5. 「次のターン」で交代
  6. 「○○勝利」で対戦終了

🎮 操作方法

  • 通常カード: フォロワー・スペル・アミュレットは同じ扱い
  • アクト: 1/3計算で別セクション
  • ターンクリア: 現在ターンの入力をリセット
  • 1ターン戻る: 最後のターンを削除

❓ トラブルシューティング

  • Pythonが見つからない → Python公式サイトからインストール
  • ポート8000が使用中 → 他のアプリを終了するか、ポート番号を変更
  • ブラウザでアクセスできない → ファイアウォール設定を確認
